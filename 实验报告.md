## 三、核心技术解析

### 3.1 CameraX 相机库

CameraX 是 AndroidX 提供的一个现代化相机库，简化了 Android 相机应用的开发。

#### CameraX 核心组件

| 组件 | 作用 |
|------|------|
| `ProcessCameraProvider` | 管理相机生命周期，将相机绑定到应用的生命周期 |
| `Preview` | 预览用例，用于显示相机实时画面 |
| `ImageAnalysis` | 图像分析用例，用于对每一帧图像进行 ML 推理 |
| `CameraSelector` | 选择前置或后置摄像头 |
| `PreviewView` | 显示相机预览的 UI 组件 |

#### 代码实现

```kotlin
private fun startCamera() {
    val cameraProviderFuture = ProcessCameraProvider.getInstance(this)

    cameraProviderFuture.addListener(Runnable {
        val cameraProvider: ProcessCameraProvider = cameraProviderFuture.get()

        // 初始化预览用例
        preview = Preview.Builder().build()

        // 初始化图像分析用例
        imageAnalyzer = ImageAnalysis.Builder()
            .setTargetResolution(Size(224, 224))  // 设置目标分辨率
            .setBackpressureStrategy(ImageAnalysis.STRATEGY_KEEP_ONLY_LATEST)
            .build()
            .also { analysisUseCase: ImageAnalysis ->
                analysisUseCase.setAnalyzer(cameraExecutor, ImageAnalyzer(this) { items ->
                    recogViewModel.updateData(items)
                })
            }

        // 选择摄像头
        val cameraSelector = if (cameraProvider.hasCamera(CameraSelector.DEFAULT_BACK_CAMERA))
            CameraSelector.DEFAULT_BACK_CAMERA else CameraSelector.DEFAULT_FRONT_CAMERA

        try {
            // 解绑所有用例
            cameraProvider.unbindAll()

            // 绑定用例到相机
            camera = cameraProvider.bindToLifecycle(
                this, cameraSelector, preview, imageAnalyzer
            )

            // 连接预览到 PreviewView
            preview.setSurfaceProvider(viewFinder.surfaceProvider)
        } catch (exc: Exception) {
            Log.e(TAG, "Use case binding failed", exc)
        }
    }, ContextCompat.getMainExecutor(this))
}
```

### 3.2 TensorFlow Lite 模型集成

#### 3.2.1 模型文件

使用 `FlowerModel.tflite` 模型文件，该模型可以识别以下花卉：
- 玫瑰 (roses)
- 雏菊 (daisies)
- 向日葵 (sunflowers)
- 郁金香 (tulips)
- 蒲公英 (dandelions)

#### 3.2.2 ML Model Binding

通过在 `build.gradle` 中启用 `mlModelBinding`，Gradle 会自动生成模型的绑定类，简化了模型的使用：

```gradle
buildFeatures {
    dataBinding = true
    mlModelBinding true  // 启用 ML 模型绑定
}
```

#### 3.2.3 模型初始化与推理

```kotlin
private val flowerModel: FlowerModel by lazy {
    // GPU 加速支持
    val compatList = CompatibilityList()
    val options = if (compatList.isDelegateSupportedOnThisDevice) {
        Log.d(TAG, "This device is GPU Compatible ")
        Model.Options.Builder().setDevice(Model.Device.GPU).build()
    } else {
        Log.d(TAG, "This device is GPU Incompatible ")
        Model.Options.Builder().setNumThreads(4).build()
    }
    FlowerModel.newInstance(ctx, options)
}

// 模型推理
val tfImage = TensorImage.fromBitmap(toBitmap(imageProxy))
val outputs = flowerModel.process(tfImage)
    .probabilityAsCategoryList.apply {
        sortByDescending { it.score }  // 按置信度降序排序
    }.take(MAX_RESULT_DISPLAY)  // 取前 3 个结果
```

### 3.3 数据视图模型 (ViewModel)

#### 架构优势

1. **生命周期感知**：ViewModel 独立于 UI 的生命周期变化（如屏幕旋转）
2. **数据持久化**：在配置更改时保留数据
3. **关注点分离**：将 UI 逻辑与数据处理分离

#### 实现方式

```kotlin
// 获取 ViewModel
private val recogViewModel: RecognitionListViewModel by viewModels()

// 观察数据变化
recogViewModel.recognitionList.observe(this, Observer {
    viewAdapter.submitList(it)  // 更新 RecyclerView
})

// 更新数据
recogViewModel.updateData(items)
```

## 七、关键技术要点

### 7.1 图像预处理

将 CameraX 获取的 YUV_420_888 格式图像转换为 Bitmap：

```kotlin
private fun toBitmap(imageProxy: ImageProxy): Bitmap? {
    val image = imageProxy.image ?: return null

    if (!::bitmapBuffer.isInitialized) {
        rotationMatrix = Matrix()
        rotationMatrix.postRotate(imageProxy.imageInfo.rotationDegrees.toFloat())
        bitmapBuffer = Bitmap.createBitmap(
            imageProxy.width, imageProxy.height, Bitmap.Config.ARGB_8888
        )
    }

    yuvToRgbConverter.yuvToRgb(image, bitmapBuffer)

    return Bitmap.createBitmap(
        bitmapBuffer, 0, 0, bitmapBuffer.width, bitmapBuffer.height,
        rotationMatrix, false
    )
}
```

### 7.2 多线程处理

使用单线程执行器处理图像分析，避免阻塞主线程：

```kotlin
private val cameraExecutor = Executors.newSingleThreadExecutor()
```

### 7.3 权限处理

```kotlin
private fun allPermissionsGranted(): Boolean = REQUIRED_PERMISSIONS.all {
    ContextCompat.checkSelfPermission(baseContext, it) == PackageManager.PERMISSION_GRANTED
}

override fun onRequestPermissionsResult(...) {
    if (allPermissionsGranted()) {
        startCamera()
    } else {
        Toast.makeText(this, getString(R.string.permission_deny_text), Toast.LENGTH_SHORT).show()
        finish()
    }
}
```
